# 新功能开发场景

这个场景展示了如何使用 Claude Code 开发新功能，从需求分析到完整实现的完整流程。

## 🎯 场景概述

### 适用情况

- 开发新的业务功能
- 实现新的 API 端点
- 创建新的用户界面
- 集成第三方服务
- 实现新的数据模型

### 核心挑战

- 需求理解和分析
- 技术方案设计
- 代码实现和测试
- 与现有系统集成
- 代码质量保证

## 📋 使用场景

### 场景 1: 用户认证功能

**需求：** 实现完整的用户认证系统，包括注册、登录、登出和密码重置。

**技术栈：** Node.js + Express + MongoDB + JWT

**主要步骤：**
1. 使用 Planner Agent 设计认证系统架构
2. 使用 TDD Guide Agent 实现认证逻辑
3. 使用 Code Reviewer Agent 审查代码
4. 使用 E2E Runner Agent 测试完整流程

**关键点：**
- JWT Token 生成和验证
- 密码哈希和验证
- 会话管理
- 安全性考虑

---

### 场景 2: 数据导出功能

**需求：** 实现数据导出功能，支持多种格式（CSV、Excel、JSON）。

**技术栈：** Python + Flask + Pandas + OpenPyXL

**主要步骤：**
1. 使用 Planner Agent 设计导出架构
2. 实现导出处理器
3. 添加异步任务处理
4. 实现进度跟踪

**关键点：**
- 大数据集处理
- 异步任务队列
- 文件生成和下载
- 错误处理和恢复

---

### 场景 3: 实时通知系统

**需求：** 实现实时通知系统，支持 WebSocket 推送和多客户端同步。

**技术栈：** Go + Gin + WebSocket + Redis

**主要步骤：**
1. 设计通知系统架构
2. 实现 WebSocket 服务器
3. 集成 Redis Pub/Sub
4. 实现客户端重连机制

**关键点：**
- WebSocket 连接管理
- 消息广播机制
- 心跳检测
- 断线重连

## 🚀 典型工作流程

### 步骤 1: 需求分析和规划

```bash
# 使用 Planner Agent
Agent(planner, """
开发用户认证功能：
- 用户注册（邮箱验证）
- 用户登录（JWT Token）
- 密码重置（邮件链接）
- 第三方登录集成（GitHub, Google）

技术要求：
- 密码使用 bcrypt 哈希
- JWT 有效期 7 天
- 支持刷新 token
- 防止暴力破解
""")
```

### 步骤 2: TDD 开发

```bash
# 使用 TDD Guide Agent
Agent(tdd-guide, """
按照 TDD 流程实现用户注册功能：
1. 先写测试（RED）
2. 实现代码（GREEN）
3. 重构代码（REFACTOR）

确保测试覆盖率达到 80% 以上。
""")
```

### 步骤 3: 代码审查

```bash
# 使用 Code Reviewer Agent
Agent(code-reviewer, """
审查认证模块代码：
- 检查安全漏洞
- 验证错误处理
- 检查性能问题
- 确保最佳实践
""")
```

### 步骤 4: 集成测试

```bash
# 使用 E2E Runner Agent
Agent(e2eater, """
测试完整的认证流程：
- 注册流程测试
- 登录流程测试
- 密码重置流程测试
- 第三方登录测试
""")
```

## 💡 最佳实践

### 1. 迭代开发

- 将大功能拆分为小任务
- 每个任务独立开发和测试
- 及时集成和验证

### 2. 持续集成

- 每个 PR 都运行完整测试
- 自动化代码质量检查
- 自动化安全扫描

### 3. 文档驱动

- 先写设计文档
- 再写 API 文档
- 最后写用户文档

### 4. 测试优先

- 单元测试覆盖核心逻辑
- 集成测试验证组件交互
- E2E 测试验证关键流程

## 🚨 常见陷阱

### 陷阱 1: 过度设计

**问题：** 过早优化，添加不必要的抽象

**解决：** YAGNI 原则 - 只实现当前需要的功能

### 陷阱 2: 忽视测试

**问题：** 只关注功能实现，忽略测试

**解决：** 使用 TDD Guide Agent 确保测试覆盖

### 陷阱 3: 硬编码

**问题：** 配置和环境硬编码

**解决：** 使用环境变量和配置文件

### 陷阱 4: 忽视安全

**问题：** 功能实现完成，但安全性不足

**解决：** 使用 Code Reviewer Agent 安全检查
