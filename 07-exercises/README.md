# 练习题集

## 概述

本目录包含 ECC 学习工程中的练习题，用于巩固和检验学习成果。练习题按难度分为初级、中级和高级，你可以根据自己的水平选择合适的练习。

## 练习分类

### 初级练习 (Beginner)

**难度**: ⭐
**适合**: 刚开始学习 ECC 的开发者
**目标**: 掌握 ECC 基本命令和概念

**练习列表**:
1. [练习 1: 使用 /plan 创建实现计划](./beginner/exercise1-plan.md)
2. [练习 2: 使用 /tdd 实现简单功能](./beginner/exercise2-tdd.md)
3. [练习 3: 使用 /code-review 审查代码](./beginner/exercise3-code-review.md)

**预计完成时间**: 每个练习 30-60 分钟

### 中级练习 (Intermediate)

**难度**: ⭐⭐⭐
**适合**: 已掌握 ECC 基础的开发者
**目标**: 掌握完整开发周期和技能应用

**练习列表**:
1. [练习 1: 完整功能开发](./intermediate/exercise1-full-feature.md)
2. [练习 2: Bug 修复完整流程](./intermediate/exercise2-bug-fix.md)
3. [练习 3: 代码重构](./intermediate/exercise3-refactoring.md)

**预计完成时间**: 每个练习 2-4 小时

### 高级练习 (Advanced)

**难度**: ⭐⭐⭐⭐⭐
**适合**: 熟练掌握 ECC 的开发者
**目标**: 掌握多代理协作和复杂场景

**练习列表**:
1. [练习 1: 多代理协作开发](./advanced/exercise1-multi-agent.md)
2. [练习 2: 完整 TDD 项目](./advanced/exercise2-tdd-full.md)

**预计完成时间**: 每个练习 4-8 小时

## 练习方法

### 推荐的学习路径

```
初级练习 1 → 初级练习 2 → 初级练习 3
    ↓
中级练习 1 → 中级练习 2 → 中级练习 3
    ↓
高级练习 1 → 高级练习 2
```

### 完成练习的步骤

1. **阅读练习说明**
   - 了解练习目标和要求
   - 查看前提条件和依赖
   - 理解预期结果

2. **准备环境**
   - 搭建开发环境
   - 准备测试项目
   - 配置必要的工具

3. **执行练习**
   - 按照练习说明执行步骤
   - 遇到问题查看提示
   - 记录解决方案

4. **验证结果**
   - 对照预期结果检查
   - 确认所有步骤完成
   - 总结学习要点

5. **记录心得**
   - 记录学到的新知识
   - 记录遇到的问题和解决方案
   - 思考如何应用到实际项目

### 练习环境

所有练习都可以在以下环境中完成：

- **Java Spring Boot Demo**: 主要的练习环境
  - 路径: `../04-practice/java-springboot-demo/`
  - 适合: 初级和中级练习

- **你自己的项目**: 用于高级练习
  - 要求: 包含完整的功能和测试
  - 适合: 中级和高级练习

## 学习建议

### 1. 循序渐进
不要跳过基础练习。高级练习依赖于中级练习的知识，中级练习依赖于初级练习的知识。

### 2. 动手实践
只看不练是学不会的。每个练习都要亲自动手完成，不要只看答案。

### 3. 记录和总结
完成练习后，记录学到的新知识、遇到的问题和解决方案。

### 4. 联系实际
思考练习中的知识如何应用到你的实际项目中。

### 5. 复习巩固
定期回顾已完成的练习，巩固学习成果。
